Download via linux

Clone the repository via CLI
```
git clone https://github.com/mcc-macca/mccmonitor
```
Go to the directory
```
cd mccmonitor
```
Take the permission for the file
```
chmod a+x ./mccmonitor
```
Now use the file
```
./mccmonitor
```
